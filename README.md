# VocabFlow (织词)

VocabFlow 是一款专为高效背单词而设计的 Web 应用程序。它集成了 3781 个精选英语单词，主要覆盖高考及高频学术词汇，并深度集成了科学的复习系统和番茄钟工具。

## ✨ 核心特性

- **📚 海量词库**: 内置 3781 个单词，每个单词均配有：
  - 音标及高质量发音
  - 核心释义
  - **常用短语 (Phrases)**: 掌握单词的地道用法
  - **词形派生 (Derivations)**: 举一反三学习形容词、副词、名词等变化
  - **精选例句 (Examples)**: 在语境中加深理解
- **♻️ 艾宾浩斯复习系统**: 基于记忆曲线自动安排复习计划，确保在遗忘临界点及时巩固。
- **⏱️ 专注番茄钟**: 内置番茄计时器，帮助你保持专注，劳逸结合。
- **📋 词库管理**: 支持词汇搜索、筛选，以及收藏夹和错题本功能。
- **🌓 主题切换**: 支持深色/浅色模式，保护视力。
- **💾 本地存储**: 所有的学习进度和个人笔记都保存在浏览器本地，无需服务器即可使用。

## 🚀 快速开始

如果你在本地使用：
1. 下载本项目所有文件。
2. 双击运行 `index.html` 即可启动。

如果你想部署到线上：
- 本项目为纯静态应用，可以直接托管至 **GitHub Pages**、**Vercel** 或 **Zeabur**。

## 🛠️ 技术栈

- HTML5 (语义化标签)
- Vanilla CSS (原生 CSS 变量、响应式设计)
- Pure JavaScript (ES6+)
- LocalStorage (本地数据持久化)

## 📁 项目结构

- `index.html`: 入口文件
- `css/style.css`: 样式文件
- `js/words.js`: 核心词库数据 (5.2MB)
- `js/vocabulary.js`: 词汇逻辑管理
- `js/spaced-rep.js`: 间隔重复算法
- `js/ui.js`: 界面交互逻辑
- `js/pomodoro.js`: 番茄钟逻辑
